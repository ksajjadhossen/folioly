# 🌟 Folioly — Curated Developer Portfolios

Folioly is a curated collection of inspiring developer portfolios from talented developers around the world. The platform helps developers discover modern portfolio ideas, UI inspirations, and real-world tech stack implementations for building their own personal websites.

Designed with simplicity and user experience in mind, Folioly provides advanced filtering, bookmarking, category-based exploration, and direct access to live portfolio websites.

---

## 🚀 Live Demo

🔗 https://folioly.vercel.app

---

## 📖 Project Overview

This project was created by studying and taking inspiration from portfolios built by high-level professionals and developers. The goal of the platform is to provide a centralized place where developers can explore outstanding portfolio designs and discover modern development trends.

Users can:

- Explore curated developer portfolios
- Search and filter portfolios efficiently
- Bookmark favorite portfolios
- Visit live portfolio websites directly
- Discover technologies used in each portfolio
- Browse categorized portfolio collections
- Switch between Dark and Light themes

Each portfolio card is fully clickable and redirects users directly to the original portfolio website for a seamless browsing experience.

---

## ✨ Features

### 🔖 Bookmark System

Save favorite portfolios directly in the browser using Local Storage.

### 🔍 Search & Filtering

Quickly discover portfolios with:

- Search functionality
- Category filters
- Advanced checkbox filtering
- Pagination support

### 🌐 External Links

Each portfolio card includes:

- Live website links
- GitHub profile links

### 🛠️ Tech Stack Display

Every card clearly displays the technologies used to build that portfolio.

### 🎨 Dark & Light Mode

Supports both Dark Mode and Light Mode for a better user experience.

### 🗂️ Categories Page

Browse all available categories and navigate to filtered portfolio collections.

### 🚀 Explore Page

Dedicated exploration page with advanced filtering options.

### ℹ️ About Page

Provides background information and details about the project.

## ⚙️ Getting Started

Clone the repository:

```bash
git clone https://github.com/ksajjadhossen/folioly.git
```

Navigate to the project directory:

```bash
cd folioly
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open your browser and visit:

```bash
http://localhost:3000
```
